== Ausgangssituation

In Österreich gibt es jährlich sehr viele Fahrschüler, die sich für einen Führerschein anmelden. In Folge möchten sie diesen dann auch abschließen.

== Istzustand

Die meisten Fahrschüler schreiben ihre Kilometer, die sie fahren müssen, Analog auf oder in Excel nieder. Dies ist zum Beispiel beim Autoführerschein nötig.


== Problemstellung

Viele Fahrschüler verlieren schnell den Überblick, oder vergessen gefahrene Kilometer einzutragen. Zudem ist die Verwaltung von vielen Zetteln sehr unübersichtlich.


== Aufgabenstellung

Es soll eine JavaFX-App entwickelt werden, die dieses Problem löst.

=== Funktionelle Anforderungen

- Der Benutzer soll neue Fahrten erfassen und speichern können (mit Datum, Kilometer, Startort, Zielort, Fahrtbedingungen)
- Der Benutzer soll Fahrten bearbeiten können, um fehlerhafte Daten zu korrigieren
- Der Benutzer soll Fahrten löschen können
- Der Benutzer soll eine Übersicht aller erfassten Fahrten erhalten
- Der Benutzer soll Fahrten nach Fahrtbedingungen (z.B. Tag/Nacht, Wetter) und Orten filtern können
- Der Benutzer soll Fahrten sortieren können (nach neuste/älteste)
- Der Benutzer soll Statistiken anzeigen können (Gesamtkilometer, Durchschnittswerte, monatliche Statistik, Wochenübersicht)
- Der Benutzer soll einen Fortschrittsbalken für den Führerschein sehen können (L17 oder Normalführerschein)
- Der Benutzer soll Fahrten importieren können (CSV/Backups)
- Der Benutzer soll Fahrten exportieren können (CSV-Format für Backups)

=== Nicht funktionale Anforderungen

- Die Daten sollen sicher und persistent verwaltet werden (H2-Datenbank mit JDBC)
- Eine intuitive Benutzeroberfläche mit zwei Tabs (Erfassung und Statistik) soll zur Verfügung stehen
- Das System soll eine sichere Datenspeicherung mit Datenbank-Persistenz bieten
- Die Bedienung soll intuitiv und einfach sein


== Ziele

- Vereinfachung der Fahrtenverwaltung für Fahrschüler
- Motivationssteigerung durch transparente Fortschrittsverfolgung
- Digitale Alternative zu analogen Methoden (Zettel, Excel)
- Transparente Verfolgung des Fortschritts zum Führerschein


== Mengengerüst

- Benutzer: 1
- In der Woche fährt man durchschnittlich 2-3 Mal, wenn man den L17 macht (das sind ca. 120 einträge im Jahr). Beim normalen Führerschein auch das Gleiche.
- Eintragungszahl kann variieren, manchmal mehr manchmal weniger.


== Rahmenbedingungen

- JavaFX (Version 21)
- Datenbank: H2
- Pattern: MVC
